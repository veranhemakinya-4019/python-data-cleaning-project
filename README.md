# Python Data Cleaning Project

## Description
This project demonstrates data cleaning using Python and pandas on a real-world messy dataset.

## Tools Used
- Python
- Pandas
- Excel

## Tasks Performed
- Loaded raw dataset
- Handled missing values
- Removed duplicates
- Renamed columns
- Saved cleaned dataset

## Outcome
This project showcases beginner-level data cleaning skills suitable for entry-level data analyst roles.
